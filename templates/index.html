Encrypt a secret message!

Use one of the three encryption techniques below to encrypt a secret message.

➡️ Offset Encryption ➡️

Encrypts your message using a simple offset on the ASCII code of each character in your message.

🤝 Shared Key Encryption 🤝

Encrypts your message using a shared key phrase. The key generates different offsets that are used on the ASCII code of each character in your message.

📇 Public Key Encryption 📇

Generate public and private keys, other people can use your public key to encrypt messages that you can decrypt with your private key.
