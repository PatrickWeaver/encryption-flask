Offset Encryption

This encrypts your message using a simple offset on the ASCII code of each character in your message.


Shared Key Encryption

This encrypts your message using a shared key phrase. The key generates different offsets that are used on the ASCII code of each character in your message.


Public Key Encryption

This helps you generate public and private keys. Other people can use your public key to encrypt messages that you can decrypt with your private key.
